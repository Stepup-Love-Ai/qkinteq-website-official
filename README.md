# Qkinteq - AI Companion Studio

Qkinteq 是一个专注于构建下一代 AI 陪伴与规划系统的项目。我们致力于为每个人打造可定制的 AI 陪伴与规划系统，让 AI 成为长期、稳定、可协作的伙伴。

## 🌟 网站功能

### 主要页面

- **首页** - 介绍 Qkinteq 的愿景、核心价值和产品方向
- **产品** - 详细展示产品线、未来规划和商业化路线
- **功能亮点** - 介绍核心功能特性
- **联系** - 获取联系方式和加入内测

### 核心特性

- 🎯 **长期关系** - 建立持续学习用户偏好的 AI 认知模型
- 💬 **一致人格** - 提供稳定的语气与情绪风格
- 📋 **结构化协作** - 将对话转化为可执行的计划和行动
- 🧠 **智能记忆** - 长期记忆与语境模型，让 AI 真正"记住你"

## 🚀 本地运行

1. 克隆仓库
   ```bash
   git clone https://github.com/Stepup-Love-Ai/qkinteq-website-official.git
   cd qkinteq-website-official
   ```

2. 启动本地服务器
   ```bash
   python3 -m http.server 8000
   ```

3. 在浏览器中访问
   ```
   http://localhost:8000
   ```
